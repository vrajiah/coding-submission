# Coding Exercises

A collection of coding problems solved in Java.

## Project Structure

```
coding-exercises/
├── app/
│   ├── src/main/java/com/exercises/   # Solution source files
│   └── src/test/java/com/exercises/   # Unit tests
├── gradle/
└── settings.gradle
```

## Tech Stack

- Java 21
- Gradle 9
- JUnit Jupiter 5

## Running Tests

```bash
./gradlew test
```

## Building

```bash
./gradlew build
```
